# AI Video Enhancer Frontend A minimal React/Next.js frontend for AI-powered image and video enhancement. ## Features - **File Upload**: Drag & drop or browse to select images/videos - **Preview**: Side-by-side comparison of original and enhanced media - **Progress Tracking**: Real-time progress bar during enhancement - **Responsive Design**: Clean UI that works on desktop and mobile ## Quick Start ```bash # Install dependencies npm install # Run development server npm run dev ``` Open [http://localhost:3000](http://localhost:3000) in your browser. ## File Structure ``` app/ ├── components/ │ ├── FileUpload.tsx # Drag & drop file upload │ ├── MediaPreview.tsx # Image/video preview │ └── ProgressBar.tsx # Enhancement progress ├── globals.css # Minimal styling ├── layout.tsx # Root layout └── page.tsx # Main page ``` ## Integration To connect with your AI backend: 1. Replace the `simulateEnhancement()` function in `page.tsx` 2. Add API calls to your enhancement service 3. Handle real file uploads and responses ## Supported Formats - **Images**: JPG, PNG, GIF, WebP - **Videos**: MP4, WebM, MOV
